확장 프로그램 개발

배경 및 목표
  • 웹 콘텐츠에서 특정 단어/용어로 독자가 독해에 어려움 발생으로 콘텐츠 만족도 저하
  • 용어 각주 제공 요청(VOC) 증가하나 지면 한계로 각주 제공 불가
  • 사용자가 브라우저에서 특정 단어를 간편 검색해 정보를 즉시 확인 가능한 확장 프로그램 개발
  • 독자들의 콘텐츠 독해력 상승 및 정보 검색 시간 단축으로 독자 편의성 증대 기대
구체화
  • 핵심 기능 개발
    • 드래그 검색 이벤트 구현
      • API 연동 및 정보 출력
        • 우리말샘 API에서 네이버 API로 교체
      • 인터페이스 공간 설정
        • IFRAME 문법에서 SIDEPANEL 형식으로 교체
      • 매니페스토 설정
        • CORS 문제 해결을 위한 퍼미션 설정
      • 크롬 로컬 스토리지 활용
  • 기능 고도화
    • 강조, 삭제, 백과사전, 북마크 페이지 한탭 이동 구현
  • 디자인 조정 및 마케팅 컨셉 설정
    • 폰트 사이즈 및 자간 행간 조정
      • 동아일보 기사 양식 참조
    • 프로그램 홍보 자료 작성
      • 효율도우미 (간편한 사전 검색 + 탭 전환 비용 감소)
결과 및 디벨롭
  • 프로토타입 개발🖼️
  • 설문 조사 자료 수집
